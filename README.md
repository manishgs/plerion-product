# Plerion product

HTTP API with Node.js and Typescript running on AWS Lambda and API Gateway using the Serverless Framework v3 and webapp with react.js.

<hr/>

## Setup API

```
cd api
cp .env.example .env
```

**Install Dependencies**

```
yarn install
```

**Run server**

```
yarn dev
```

**Deploy**

```
yarn deploy
```

<hr/>

## Setup Web

```
cd web
cp .env.example .env
```

**Install Dependencies**

```
yarn install
```

**Run app**

```
yarn dev
```
